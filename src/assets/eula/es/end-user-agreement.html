Uso del acuerdo de usuario final del software e-Mobility

Datos personales

Solo se almacenarán los datos sobre el uso de las estaciones de carga por usuario:

  • Nombre y apellidos
  • Email
  • Teléfono, número de móvil
  • ID de insignia
  • Estatud de cuenta (activa, bloqueada...)
  • Roles de usuario (básico, admin...)
  • Idioma
  • Fotografía
  • Dirección
  • Información del vehículo (Número de placa, VIN...)
  • Contraseña (hashed)
  • Número de identificación profesional
  • Centro de costos profesionales

Datos de usuario comercial

  • Consumo instantáneo y total
  • Estatus de las estaciones de carga
  • Empezar/Detener sesiones
  • Autorizaciones
  • Curvas de carga (Datos de consumo)
  • Curvas de costo (Datos de precio)
  • Datos de sesiones (fecha/hora, duración, inactividad, consumo total...)
  • Sesiones activas
  • Historial de sesiones
  • Asignación de sitios
  • Vehículos
  • Información de facturación
  • Registros del sistema

Escenarios

  • Análisis (uso, consumo, costo ...)
  • Gestión del sitio
  • Gestión de autorizaciones
  • Gestión de precios
  • Gestión de facturación
  • Carga inteligente
  • Reserva de estación de carga
  • Predicciones de carga
  • Notificaciones de usuarios y administradores
  • Carga en casa
  • Reembolso de sesiones
  • Roaming

Objetivo del procesamiento de datos

  • Optimice el uso de las estaciones de carga por parte de los usuarios manteniendo el costo bajo.
  • Mejore el mantenimiento y la rotación del vehículo en las estaciones de carga.
  • Desarrolle un algoritmo de optimización de carga para introducir un sistema de gestión de carga que permita a los vehículos eléctricos cargar concurrentemente de acuerdo con las restricciones del sitio (potencia máxima del sitio ...)
  • Aliente a los usuarios a cargar en casa para reducir la carga de la infraestructura y, finalmente, obtener el reembolso de sus consumos de electricidad por cargar sus autos profesionales

Privacidad

  • Esta aplicación ha sido diseñada teniendo en cuenta toda la privacidad de los datos personales desde el principio.

Integraciones de productos de terceros

  • SAP Analytics: utilizado para procesar y mostrar estadísticas complejas
  • SAP Concur: ulitizado para reembolsar la energía gastada por los usuarios en casa.
  • SAP Convergent Charging/Invoicing: utilizado para realizar precios complejos en una transacción
  • Stripe: utilizado para la facturación y el pago de las sesiones.
  • Gireve / Hubject: utilizado para publicar las estaciones de carga en una plataforma móvil
  • Google Firebase: utilizado para enviar notificaciones push remotas a los teléfonos móviles del usuario

Usuarios involucrados

  • Los usuarios que hacen uso de las estaciones de carga en el sitio monitoreado y que se registraron en el e-Mobility aplicación.

Objetivo de procesamiento de datos

  • El procesamiento de datos es para administradores de sitios, científicos de datos y demostraciones para clientes internos / externos.
  • Los usuarios acceden a los datos solo a través del protocolo seguro (HTTP)
  • Los científicos de datos de SAP reciben solo las transacciones anónimas de un administrador para probar sus algoritmos de carga.
  • Para demostrar el e-Mobility aplicación para personas externas (no SAP), utilizamos un usuario demo para anonimizar todos los datos personales.
  • Las sesiones de los usuarios pueden ser enviadas a los sistemas de SAP Analytics en Alemania con fines de prueba o demostraciones.
  • Las sesiones del usuario pueden ser procesadas para predecir la demanda de cobro en un sitio determinado.

Datos de localización

  • La aplicación se implementará y los datos se almacenarán en una plataforma en la nube ubicada en Europa.

Derecho al olvido

  • Sus datos personales se conservarán hasta el final de su contrato de trabajo con la empresa.
  • Sus datos personales se borrarán manualmente después de un período de inactividad de 6 meses (sin conexión a la aplicación durante ese tiempo).
  • Sus datos personales se borrarán en caso de que lo solicite (consultelo en Oficial de privacidad de datos)

Seguridad

  • Se requiere usuario y contraseña para acceder a la aplicación para todos los usuarios.
  • Los datos se envían de forma segura a través de la red mediante un protocolo cifrado (HTTP), lo que garantiza la confidencialidad.
  • Solo los administradores están autorizados a leer, escribir, editar, eliminar todos los datos almacenados en la base de datos.
  • Los usuarios estándar solo están autorizados a leer y editar sus propios datos personales, ver la disponibilidad de las estaciones de carga, el cargo actual, la historia de sus sesiones y sus estadísticas.
  • Sus datos pueden ser comunicados a terceros en caso de consulta legal como "Inspección de trabajo" o "servicios fiscales" o servicios policiales.

Consentimiento del usuario

  • Debe dar su consentimiento explícito para que usemos sus datos.
  • Esto se realiza a través de una casilla de verificación en la página de inicio de sesión del e-Mobility aplicación.

Derecho de acceso y corrección de datos.

  • Usted puede, previa prueba de su identidad, realizar una consulta al Oficial de privacidad de datos o al administrador sobre el derecho de acceder a sus datos personales.
  • Puede solicitar al Oficial de privacidad de datos, que la información sobre sus datos que es inexacta, está incompleta, equivocada, ya no es válida, o que el uso, comunicación o mantenimiento de estos está prohibido, puede corregirse, completarse, aclararse, actualizarse o eliminarse.

Derecho a objetar

  • Tiene derecho a oponerse al uso de todos o parte de sus datos contactando a Oficial de privacidad de datos.

Limite el procesamiento de datos

  • Tiene derecho a limitar parte del uso de sus datos.

Derecho de portabilidad.

  • Puede solicitar, alOficial de privacidad de datos, la transferencia de todos o parte de sus datos personales en un archivo que sea legible.
  • También puede exportar todas o parte de sus sesiones en la aplicación.
